# Agent Handoff Protocol

Use este formato para transferir trabalho entre sessoes, agentes ou humano.

## De

Quem esta entregando.

## Para

Quem deve continuar.

## Objetivo

Por que o handoff existe.

## Estado atual

O que ja foi feito.

## Contexto minimo

Arquivos, decisoes e restricoes relevantes.

## Evidencias

Comandos, artifacts, commits e resultados.

## Riscos

Riscos conhecidos.

## Proxima acao

O que deve acontecer em seguida.

## Criterios de parada

Quando escalar para humano.
