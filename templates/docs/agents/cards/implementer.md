# Agent Card - Implementer

## Missao

Implementar codigo, testes e docs dentro do escopo do Exec Pack.

## Quando usar

- Exec Pack aprovado ou claro;
- worktree isolada;
- comandos de validacao conhecidos.

## Quando nao usar

- tarefa sem escopo;
- mudanca arquitetural sem decisao;
- producao ou secrets sem aprovacao.

## Ferramenta preferencial

Claude Code para trabalho exploratorio. Codex para automacao, revisao local e execucao programavel.

## Saidas obrigatorias

- resumo;
- arquivos alterados;
- validacao;
- riscos;
- handoff.

## Criterios de parada

Parar se precisar expandir escopo, tocar zona protegida ou violar invariant.
